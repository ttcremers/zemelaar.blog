# Editorial rhythm pass notes

This branch applies the approved Zemelaar rhythm-pass style to the remaining migrated posts.

Scope:

- Break long paragraphs into shorter reading blocks.
- Split long thought-blocks where the rhythm benefits from a pause.
- Let punchlines and transitions breathe.
- Mark obvious standalone quotations as blockquotes.

Guardrails:

- Do not rewrite sentences for style.
- Do not change jokes, tone, meaning, titles, slugs, front matter, links, or images.
- Do not use a script for the editorial pass.

Review suggestion:

Read the rendered pages first, then inspect the diff only where a passage feels too choppy or too sparse.

```text
Generator: Hugo

```

## Design direction

Zemelaar should feel like a quiet European literary archive.

The design should be restrained, text-first, and typographic.

Avoid decorative writerly effects, handwritten fonts, typewriter nostalgia, dashboard aesthetics, and tech-blog patterns.
